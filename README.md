# 🪃 ブーメランチェッカー

国会議事録APIとGemini APIを使って、国会議員の過去の矛盾・ブーメラン発言を自動検出するCLIツールです。

## 仕組み

1. [国会会議録検索システムAPI](https://kokkai.ndl.go.jp/api.html)から指定した議員の発言を取得
2. Gemini API で発言間の矛盾を分析・スコアリング（原文からの一字一句抜粋も返させる）
3. **弁護人レビュー（切り抜き防止の検証層）**: 検出された各ペアについて
   - 議事録の**前後の質疑文脈**と発言者の肩書を取得し、
   - 「政治家側の弁護人」の視点で反論を全力で検討する2段目のLLMを通す
   - 誤読（他者発言の引用・仮定の話等）→ **棄却** / 状況変化等で説明がつく → **言い分付きで保持** / 弁護不成立 → **矛盾確定**
   - あわせて、LLMが返した抜粋が**議事録原文と一致するかをコードで機械検証**（捏造・改変引用の排除）
4. 結果をターミナルに表示（SNS投稿用フォーマットにも対応。**SNS/X向け出力は「矛盾確定＋原文一致」のものだけ**が対象になる）

## セットアップ

### 前提条件

- Python 3.10 以上
- [Gemini API キー](https://aistudio.google.com/apikey)（無料枠で動作。日次リクエスト数はツール側で自動制限）

### インストール

```bash
git clone https://github.com/your-repo/boomerang-checker.git
cd boomerang-checker
pip install -r requirements.txt
```

### 環境変数の設定

```bash
export GEMINI_API_KEY="your-api-key-here"
```

## 使い方

### 基本的な使い方

```bash
# 議員名を引数で指定
python main.py 岸田文雄

# 対話的に議員名を入力
python main.py
```

### オプション

```bash
# 取得する発言数を指定（デフォルト: 50）
python main.py 岸田文雄 --max-speeches 100

# 日付範囲を指定
python main.py 岸田文雄 --from-date 2020-01-01 --until-date 2024-12-31

# 争点キーワードで絞り込み（Xで炎上中の争点に対する立場変化を照合する使い方。
# 本文検索で絞れる分、10〜30年前の「昔の約束」まで遡って取得する）
python main.py 野田佳彦 --keyword 定数削減

# SNS投稿用フォーマットも表示
python main.py 岸田文雄 --sns

# 弁護人レビュー（文脈検証）をスキップ（API使用量の節約用。SNS出力の信頼性は下がる）
python main.py 岸田文雄 --no-verify

# APIキーを直接指定
python main.py 岸田文雄 --api-key xxxxx
```

## プロジェクト構成

```
boomerang-checker/
├── main.py                  # CLIエントリーポイント
├── requirements.txt         # 依存パッケージ
├── README.md
└── boomerang/
    ├── __init__.py
    ├── kokkai_api.py        # 国会会議録API クライアント（発言取得＋前後文脈取得）
    ├── analyzer.py          # Gemini API 矛盾分析（検出＋原文抜粋の機械検証）
    ├── verifier.py          # 弁護人レビュー（文脈検証・切り抜き防止）
    └── formatter.py         # 出力フォーマッター
```

## 使用API

- **国会会議録検索システム API**: https://kokkai.ndl.go.jp/api.html
  - 国立国会図書館が提供する無料API
  - 第1回国会（1947年）からの全会議録を検索可能
- **Gemini API**: https://ai.google.dev/
  - 発言間の矛盾検出・スコアリング、および弁護人レビューに使用

## 注意事項

- 国会会議録APIは無料ですが、サーバー負荷軽減のためリクエスト間隔を設けています
- 矛盾スコアはAIによる推定値です。弁護人レビューと原文照合で誤検出を減らしていますが、公開前には必ず出典リンクから原文を確認してください
- SNS向け出力（--sns / --x）は、検証を通過した「矛盾確定」ペアのみを対象とします（切り抜き・文脈無視の指摘を受けないため）
- 本ツールは議論・分析目的であり、特定の政治的立場を支持するものではありません
